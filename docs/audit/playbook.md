## 1. 概述

本文件存放审计 agent 的累积型实战经验（见[章程](./CHARTER.md) §14.2）。每周期开始时与章程一同读入；复盘时 agent 在此追加或整理条目。

- agent 可自由读写本文件；影响规则 / 范围 / 契约的结构性认知则直接改 `CHARTER.md` 相应章节。
- 每条**必须**可溯源（注明触发它的 `run` 与 `finding`），呼应 `EVOLVE-EVIDENCE`。
- 本文件随 `docs/audit/**` 一并由 git 兜底：agent **不 commit**，人工审阅 diff 后决定提交或回退（`EVOLVE-NO-COMMIT`）。
