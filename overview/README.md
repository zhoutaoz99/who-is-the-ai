# overview · 系统全景导览（可视化页）

一个**自包含、零构建**的可视化解读页，用「逐层放大 + 滚动叙事」的方式讲清《谁是AI》整套系统：
从在线 ⇄ 离线双回路的全局闭环，到离线优化流水线每一站的设计意图，再到三道防自欺闸门、九条核心洞察与各模块深潜。

## 怎么看

直接用浏览器双击打开 `index.html` 即可，无需启动任何服务、无需安装依赖。

```
overview/
  index.html        页面骨架 + 全部内联 SVG 图
  assets/style.css  视觉编码与排版（复用产品品牌色：暖纸 + 青绿）
  assets/app.js     滚动叙事引擎 / 流水线小车 / 卡片交互（纯原生 JS，无第三方库）
```

## 页面结构（自上而下）

| 区块 | 看什么 |
|---|---|
| Hero | 一句话目标 + 信念 + 唯一成败指标 + 颜色图例 |
| L0 双回路 | 在线 ⇄ 离线互喂闭环，滚动驱动高亮 |
| L1 三层架构 | 在线层 / 离线优化层 / 契约层 / 参与方 |
| L2 离线流水线 | 场景库→引擎→裁判→★聚合→优化器→编排器，小车随滚动前进 |
| 三道闸门 | 配对+显著性 / 留出集+探测轮换 / 真人校准（棘轮隐喻）|
| 九条洞察 | 反直觉判断 → 下游设计（点开卡片）|
| 模块深潜 | 每个模块：是什么 / 关键契约 / 为什么 / 代价（折叠卡）|
| 文档地图 | 链接回 `docs/` 下的权威设计文档与推荐阅读路径 |
